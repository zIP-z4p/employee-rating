# Структура проекта и архитектурные решения

## 1. Назначение документа
Документ описывает:
- структуру проекта (backend + frontend + инфраструктура);
- основные архитектурные решения и их обоснование;
- ключевые принципы масштабируемости и поддержки.

---

## 2. Общая архитектура

Система реализована как набор контейнеризованных сервисов:

- **Backend** (Python/FastAPI) — REST API, бизнес-логика расчёта рейтингов, импорт данных.
- **PostgreSQL** — основное хранилище сущностей и истории оценок.
- **Redis** — кэширование готовых снимков рейтинга и ускорение ответов API.
- **Frontend** (React/TypeScript/Vite) — веб-дашборд для просмотра рейтингов и трендов.
